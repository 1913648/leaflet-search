Leaflet.Control.Search

A Leaflet Control for search markers location by attribute
and much more...

Features

  • Autocomplete
  • Retrieve data locations through Ajax/Jsonp
  • Pre-filtering data from Ajax/Jsonp
  • Search callback support
  • Localization placeholder and text alert
  • Autozoom on location founded
  • Autoresize textbox

Demos

  • Simple
  • Ajax
  • Jsonp
  • Jsonp Filtered
  • Bulk data
